    # Theme Events

    The **theme events** are responsible for managing the theme direction (left to right or right to left) within the application.<br/>
    This document presents information for the event offered by **``@ufe/base``** module, as well as examples of how it can be used.

    The events described in this document are: 

    - [Changing the Theme Direction](#changing-the-theme-direction)
    - [Changing the Theme Palette](#changing-the-theme-palette)

    <br/>
    <br/>

    ## Changing the Theme Direction

    The **changeThemeDirection** function in ``@ufe/base`` module is responsible for dispatching the event ``ufe-change-theme-direction``.<br/> 
    This event is used for changing the theme direction from left to right (LTR) or right to left (RTL).

    The function has the following parameters: 
    <table>
        <thead>
            <tr>
                <th>Name</th>
                <th>Description</th>
                <th>Type</th>
                <th>Default</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>**direction***</td>
                <td> The theme direction. </td>
                <td> ```'ltr' | 'rtl'``` </td>
                <td> - </td>
            </tr>
            <tr>
                <td>**dontSetUserPrefs**</td>
                <td> Boolean to control whether to set or not user preferences. </td>
                <td> ```boolean``` </td>
                <td> false </td>
            </tr>
        </tbody>
    </table>

    <br />

    ### Code Example

    Below is an example of how to change the theme direction.

    ```js
    import { changeThemeDirection } from '@ufe/base';

    // To change the theme direction to RTL
    changeThemeDirection('rtl');

    // To change the theme direction to LTR without setting user preferences
    changeThemeDirection('ltr', true);
    ```

    ## Changing the Theme Palette

    The **changeThemeType** function in ``@ufe/base`` module is responsible for dispatching the event ``ufe-change-theme-type``.<br/> 
    This event is used for changing the theme palette from light to dark or dark to light.

    The function has the following parameters: 
    <table>
        <thead>
            <tr>
                <th>Name</th>
                <th>Description</th>
                <th>Type</th>
                <th>Default</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>**type***</td>
                <td> The palette type. </td>
                <td> ```'light' | 'dark'``` </td>
                <td> - </td>
            </tr>
            <tr>
                <td>**dontSetUserPrefs**</td>
                <td> Boolean to control whether to set or not user preferences. </td>
                <td> ```boolean``` </td>
                <td> false </td>
            </tr>
        </tbody>
    </table>

    <br />

    ### Code Example

    Below is an example of how to change the theme palette.

    ```js
    import { changeThemeType } from '@ufe/base';

    // To change the theme type to light
    changeThemeType('light');

    // To change the theme type to light without setting user preferences
    changeThemeType('light', true);
    ```
