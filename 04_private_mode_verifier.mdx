    # Private Mode Verifier

    The **Private Mode Verifier** component is used to conditionally render its children based on the application's private mode state.<br/>
    It provides two modes of behavior, hiding the children or blurring them when the application is set to private.

    We present in the code below an example of how to import and use the component.

    ```jsx
    import React from 'react';
    import { PrivateModeVerifier } from '@ufe/base';
    
    function App() {
        return (
            <PrivateModeVerifier 
                forbidden='blur'>
                {children}
            </PrivateModeVerifier>
        );
    }
    ```

    <br/>
    <br/>
