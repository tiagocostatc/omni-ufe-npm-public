    # User Settings Events
    
    **User Settings events** are responsible for managing the state of the overlay and quick actions, as well as refreshing the session timeout.
    This document presents information for each of the events offered by ```@ufe/base``` module, as well as examples of how they can be used.

    The events described in this document are:

    - [Set Overlay Open](#set-overlay-open)
    - [Set Quick Action Open](#set-quick-action-open)
    - [Refresh Session Timeout](#refresh-session-timeout)

    <br/>
    <br/>

    ## Set Overlay Open

    The **setOverlayOpen** function in ```@ufe/base``` module is responsible for dispatching the event **```user-settings-set-overlay-open```**.
    This event is responsible for changing the user settings overlay state.

    The function has the following parameters: 
    <table>
        <thead>
            <tr>
                <th>Name</th>
                <th>Description</th>
                <th>Type</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>**value***</td>
                <td>User settings overlay state to be set.</td>
                <td> ```boolean``` </td>
            </tr>
        </tbody>
    </table>

    <br/>

    ### Code example

    In the example below we show how to change the ufe settings overlay state.

    ```javascript
    import { setOverlayOpen } from '@ufe/base';

    export default function myFunction(){
        setOverlayOpen(true);
    }
    ```

    <br/>
    <br/>

    ## Refresh Session Timeout

    The **refreshSessionTimeout** function in ```@ufe/base``` module is responsible for dispatching the event **```ufe-refresh-session-timeout```**.
    This event is responsible for refreshing ufe session timeout on user settings.

    <br/>

    ### Code example

    In the example below we show how to refresh ufe session timeout.

    ```javascript
    import { refreshSessionTimeout } from '@ufe/base';

    export default function myFunction(){
        refreshSessionTimeout();
    }
    ```
