# Ellipsis Tooltip

This section contains an exemple of Ellipsis Tooltip already pre built and ready to use.

The **EllipsisTooltip** component is a TextField that, when exceeding the predefined limit, show an ellipsis and displays a tooltip with the total content of the string.

These component was created to be easier to use, with less props and for more direct use cases.
