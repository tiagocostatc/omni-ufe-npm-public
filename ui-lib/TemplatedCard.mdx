# Templated Card

The **Template Card** is a subset of Cards already customized with a specific layout. It's possible to use the card with a title, subtitle, content, and actions.

### Code

```jsx

import React from 'react';
import { TemplatedCard } from '@uilib/components';
import { TrashO, ArrowRight } from '@uilib/icons';
```
