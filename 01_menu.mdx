    # Menu Events

    The **menu events** are responsible for opening/closing the menu, collapsing/expanding menu categories and changing the menu view mode.<br/> 
    This document presents information for each of the events offered by **``@ufe/base``** module, as well as examples of how they can be used.

    The events described in this document are: 

    - [Opening/Closing the Menu](#openingclosing-the-menu)
    - [Collapsing a Category](#collapsing-a-category)
    - [Expanding a Category](#expanding-a-category)
    - [Collapsing All Categories](#collapsing-all-categories)
    - [Expanding All Categories](#expanding-all-categories)
    - [Changing Menu View Mode](#changing-menu-view-mode)

    <br/>
    <br/>

    ## Opening/Closing the Menu

    The **openMenu** function in ``@ufe/base`` module is responsible for dispatching the event ``ufe-open-menu``.<br/> 
    This event is used for opening or closing the application's menu.

    The function has the following parameters: 
    <table>
        <thead>
            <tr>
                <th>Name</th>
                <th>Description</th>
                <th>Type</th>
                <th>Default</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>**open***</td>
                <td> Indicates whether to open (`true`) or close (`false`) the menu. </td>
                <td> ```boolean``` </td>
                <td> - </td>
            </tr>
        </tbody>
    </table>

    <br />

    ### Code Example

    In the example below we show how to open and close the menu.

    ```js
    import { openMenu } from '@ufe/base';

    // To open the menu
    openMenu(true);

    // To close the menu
    openMenu(false);
    ```

    <br />
    <br />

    ## Collapsing a Category

    The **collapseCategory** function in ``@ufe/base`` module is responsible for dispatching the event ``ufe-collapse-category``.<br/> 
    This event is used for collapsing a specific category in the menu.

    The function has the following parameters: 
    <table>
        <thead>
            <tr>
                <th>Name</th>
                <th>Description</th>
                <th>Type</th>
                <th>Default</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>**category***</td>
                <td> Category to collapse. </td>
                <td> ```string``` </td>
                <td> - </td>
            </tr>
        </tbody>
    </table>

    <br />

    ### Code Example

    In the example below we show how to collapse the "Business" category.

    ```js

    // To collapse the "Business" category
    collapseCategory('Business');
    ```

    ## Changing Menu View Mode

    The **changeMenuView** function in ``@ufe/base`` module is responsible for dispatching the event ``ufe-change-menu-view-mode``.<br/> 
    This event is used for changing the menu view mode to either a grid or list display.

    The function has the following parameters: 
    <table>
        <thead>
            <tr>
                <th>Name</th>
                <th>Description</th>
                <th>Type</th>
                <th>Default</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>**mode***</td>
                <td> Menu view mode. </td>
                <td> ```MENU_LAYOUT_GRID'` or `'MENU_LAYOUT_LIST``` </td>
                <td> - </td>
            </tr>
        </tbody>
    </table>

    <br />

    ### Code Example

    In the example below we show how to change the menu view mode to grid and list.

    ```js
    import { changeMenuView } from '@ufe/base';

    // To change the menu view to grid mode
    changeMenuView('MENU_LAYOUT_GRID');

    // To change the menu view to list mode
    changeMenuView('MENU_LAYOUT_LIST');
    ```
