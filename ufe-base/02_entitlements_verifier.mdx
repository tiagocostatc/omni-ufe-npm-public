    # Entitlements Verifier
    
     The **Entitlement Verifier** component validates if the user, given a set of entitlements and a comparator, has the necessary entitlements to render the child components.
     
     If the comparator is of type **```'or'```**, the user simply needs to have one of the entitlements to render the component. <br/>
     If the comparator is of type **```'and'```**, the user must have all entitlements to render the component.


    We present in the code below an example of how to import and use the component.

    ````javascript
    import { EntitlementsVerifier } from '@ufe/base';

    <EntitlementsVerifier 
        entitlements=["OPEN_PRODUCTS", "OPEN_ADMIN"] 
        comparator="or">
        {children}
    </EntitlementsVerifier>
    ````

    <br/>
    <br/>
