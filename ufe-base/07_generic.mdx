    # Generic Events
    
    **Generic events** are responsible for executing generic UFE events like language change.
    This document presents information for each of the events offered by ```@ufe/base``` module, as well as examples of how they can be used.

    The events described in this document are:

    - [Change Language](#change-language)

    - [Set Loading State](#set-loading-state)

    <br/>
    <br/>

    ## Change Language

    The **changeLanguage** function in ```@ufe/base``` module is responsible for dispatching the event **```ufe-change-language```**.
    This event is responsible for changing the current language being used in the Unified Frontend.

    The function has the following parameters: 
    <table>
        <thead>
            <tr>
                <th>Name</th>
                <th>Description</th>
                <th>Type</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>**language***</td>
                <td>The target language (should exist in the languages manifest, by default ```pt-PT``` and ```en-US```)</td>
                <td> ```string``` </td>
            </tr>
        </tbody>
    </table>

    <br/>

    ### Code example

    In the example below we show how to change UFE current language.

    ```javascript
    import { changeLanguage } from '@ufe/base';

    export default function myFunction(){
        changeLanguage('pt-PT');
    }
    ```

    ## Set Loading State

    The **setLoadingState** function in ```@ufe/base``` module is responsible for dispatching the event **```ufe-set-loading-state```**.
    This event is responsible for setting and stopping the loaders in Unified Frontend.

    The function has the following parameters: 
    <table>
        <thead>
            <tr>
                <th>Name</th>
                <th>Description</th>
                <th>Type</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>**value***</td>
                <td> The value that controls if the loading should be enabled or stopped. </td>
                <td> ```boolean``` </td>
            </tr>
            <tr>
                <td>**target***</td>
                <td> The target of the loading animation, either the application or ufe itself. </td>
                <td> ```application | ufe``` </td>
            </tr>
            <tr>
                <td>**message***</td>
                <td> A message to be displayed below the loading animation. </td>
                <td> ```string``` </td>
            </tr>
        </tbody>
    </table>

    <br/>

    ### Code example

    In the example below we show how to use the UFE set loading state.

    ```javascript
    import { setLoadingState } from '@ufe/base';

    export default function myFunction(){
        setLoadingState(true,'application', 'Test');
    }
    ```
